# API Python S-exp

Пользователь взаимодействует только с объектной моделью
AST (Abstract Syntax Tree).

## Функции API

Публичный интерфейс включает две функции:

- `loads` — преобразование строкового представления S-выражения в объектное AST
- `dumps` — преобразование объектного AST обратно в строковое S-выражение

Данные функции являются взаимно обратными для корректных входных данных:
`dumps(loads(x))` возвращает строковое представление, эквивалентное исходному
S-выражению.

---

## loads

```python
loads(text: str) -> Node
```

Преобразует строку, содержащую S-выражение, в объектное представление
в виде абстрактного синтаксического дерева (AST).

Функция выполняет лексический и синтаксический анализ входной строки и
возвращает корневой узел дерева, описывающего структуру данных.

### Параметры
- `text (str)` — входная строка в формате S-выражения

### Возвращаемое значение
- `Node` — корневой узел AST

### Пример использования
```python
from src.api.api import loads

ast = loads('(person (:name "Alice") (:age 30))')
```

---

## dumps

```python
dumps(node: Node) -> str
```

Преобразует объектное представление абстрактного синтаксического дерева
(AST) обратно в строку в формате S-выражения.

Функция использует метод сериализации, определённый для узлов AST, и
гарантирует корректное формирование выходного выражения при условии
корректности входного дерева.

### Параметры
- `node (Node)` — корневой узел AST

### Возвращаемое значение
- `str` — строковое представление S-выражения

### Пример использования
```python
from src.api.api import dumps

text = dumps(ast)
```
